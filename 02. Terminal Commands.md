# Terminal Commands

Note: Powershell sometimes get problematic for me, so i prefer using Command prompt in Terminal

```
:: 1. Compile
avr-gcc -mmcu=atmega8 -Os -o output.elf try.c

:: 2. Convert to hex
avr-objcopy -O ihex output.elf output.hex

:: 3. Flash
avrdude -c arduino -p m8 -P COM3 -b 19200 -U flash:w:output.hex:i

-REPLACE-
Arduino Nano (old bootloader) → 19200
Arduino Nano (new bootloader) → 115200
Arduino Uno → 115200

```

here,
- wherever `try` --> rename as the `.c` file name
- wherever `output` --> rename as preferred to create `.hex` file


# Errors

- `0xffffff` = no connection at all
- `0x000000` = connection exists but something is still wrong
